# A. CPCS

## 题目描述

在上一个宝箱的密码被成功破解后，小鸡大惊失色，连夜更换了密码的生成规则，并重新将宝箱锁好，信心十足地摆在了所有参赛者的面前。  

这一次，解锁的线索更加复杂 —— 宝箱表面刻着两串小写字母 $a$ 和 $b$ ，并附言：“若你能算清 $a$ 的子序列中有多少与 $b$ 相同，我便甘拜下风！”  

现在，宝箱的最终命运掌握在你手中！  

特别地，如果一个字符串可以通过删除另一个字符串中的若干个字符（可能为零个），且不改变其余字符的顺序得到，那么这个字符串被称为另一个字符串的子序列。例如， $\texttt{ace}$ 是 $\texttt{abcde}$ 的子序列，但 $\texttt{aec}$ 不是。  


## 输入描述

第一行包含一个字符串 $a$  $(1 \le \lvert a \rvert \le 10^5)$ 。  

第二行包含一个字符串 $b$  $(1 \le \lvert b \rvert \le 10^3)$ 。  


## 输出描述

输出一个整数，表示与 $b$ 相同的 $a$ 的子序列数。  

由于答案可能很大，请将结果对 $998244353$ 取模。  


## 样例

### 样例 1
**输入**:
```
scmupc
sc
```

**输出**:
```
2
```

**说明**:  

对于第一组样例， $a$ 中共有两个子序列与 $b$ 相同，用方框标记，分别为 $\mathtt{\fbox{sc}mupc}$ 和 $\mathtt{\fbox{s}cmup\fbox{c}}$ 。  
  


### 样例 2
**输入**:
```
scppcsscpspccsp
scpc
```

**输出**:
```
28
```
