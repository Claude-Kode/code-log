1. 定义 —— 直径 = 树上最长简单路径长度（节点数或边权和）
2. 核心结论（3 条，含简要证明思路）
3. 解法一：两遍 DFS/BFS（适用于边权非负）
- 任取一点 → 找最远点 A → 从 A 找最远点 B → AB 即直径
- O(n)，只需邻接表
4. 解法二：树形 DP（适用于任意边权，含负权）
- 每个点维护向下的最长路径 + 次长路径（不同子树）
- ans = max(最长 + 次长)，O(n)

结论 1　从树上任意一点出发，距离它最远的点一定是某条直径的一个端点  

结论 2　由结论 1 可得——两遍 DFS/BFS 即可求出直径：第一遍从任意点找最远点 A，第二遍从 A 找最远点 B，AB 即为直径。

结论 3　树的中心（直径中点）到所有节点的最大距离最小。如果需要找一个点使最远距离最小化，答案在直径中点

```
直径的最高点一定在某处"拐弯"，那个拐弯点就是 LCA(端点A, 端点B)。DP 枚举了所有点做拐弯点，每个点都能看到它所有儿子的最长向下路径，所以一个都不会漏。
```

注意, 简单路径的本质上其实是靠 路径的节点计数来进行计算的

中心让"最远距离"最小 → 跟直径有关  
1. 中心一定在直径上
2. 设直径长度为 D，中心在直径的中点位置
3. D 为偶数 → 恰好 1 个中心（某个节点上）
4. D 为奇数 → 恰好 2 个中心（直径中点落在一段边上，两边端点是中心）

重心让"最大连通块"最小 → 跟子树大小有关
1. 一棵树最多 2 个重心
2. 如果有 2 个重心，它们一定相邻 (一条边的两端)
3. 以重心为根，所有子树大小 ≤ n/2
4. 重心到所有节点的距离和最小 (不是最大距离，是总和)
5. 点分治的核心：每层选重心，保证每层规模至少减半 → 递归深度 ≤ log n
