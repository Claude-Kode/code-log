# 图的基本术语知识
## 基础图结构

• 子图 (Subgraph): 点集和边集均为原图点集和边集子集的图.

- 导出子图 (Induced Subgraph): 对于一个子图来说, 子图内部的互相关系被完整的保留下来, 就是一个导出子图.

## 团与覆盖

• 团 (Clique): 完全子图.

• 最大团 (Maximum Clique): 顶点数最多的团.

- 最小团 (Minimum Clique): 通常指大小为 1 的团 (单个顶点).
• 极大团 (Maximal Clique): 不是其他团的子图的团 (即无法再添加顶点而保持完全性).

- 最小团覆盖 (Minimum Clique Cover): 用最少的团覆盖所有的点, 所需团的数量记为 \kappa(G).

## 染色与度序列

• 最小染色 (Minimum Coloring): 用最少的颜色对图进行完整的染色 (相邻顶点颜色不同).

- 可图化序列 (Graphic Sequence): 如果给定一个序列 a, 可以找到一个图 G 以其为度数列, 则称 a 是可图化的.

## 强连通性 (有向图)

• 强连通 (Strongly Connected): 有向图 G 中任意两个结点互相可达

• 强连通分量 (SCC): 极大的强连通子图 (Strongly Connected Component)

• 关于极大的定义: 在可选关系中添加任何一个都会破坏原来所维系的关系.

## 连通性与割 (无向图)

• 割点 (Cut Vertex / Articulation Point): 删除该顶点后, 图的连通分量数量增加.

- 割边 / 桥 (Bridge): 删除该边后, 图的连通分量数量增加.
• 双连通分量 (Biconnected Component, BCC):

  - 点双连通分量: 不含割点的极大连通子图 (任意两点至少存在两条点不重复的路径).
  - 边双连通分量: 不含桥的极大连通子图 (任意两点至少存在两条边不重复的路径).

## 集合与覆盖 (极值图论)

• 支配集 (Dominating Set): 一个顶点子集 D, 使得图中每个顶点要么在 D 中, 要么与 D 中的某个顶点相邻.

• 独立集 (Independent Set): 一个顶点子集 I, 其中任意两个顶点都不相邻.

• 匹配 (Matching): 一个边集 M, 其中任意两条边没有公共顶点.

• 点覆盖 (Vertex Cover): 一个顶点子集 C, 使得图中每条边至少有一个端点在 C 中.

• 边覆盖 (Edge Cover): 一个边集 E', 使得图中每个顶点都至少是 E' 中某条边的端点.

## 极值概念对比

概念 定义核心 极值问题

最大独立集 顶点互不相邻 求顶点数最多的独立集

最大匹配 边无公共顶点 求边数最多的匹配

最小点覆盖 覆盖所有边 求顶点数最少的点覆盖

最小边覆盖 覆盖所有顶点 求边数最少的边覆盖

注: 在二分图中, 最大匹配数 = 最小点覆盖数 (König 定理).
