另外介绍一下 pass

pass 一般用于占位置。
在 Python 中有时候会看到一个 def 函数:

def sample(n_samples):
    pass

该处的 pass 便是占据一个位置，因为如果定义一个空函数程序会报错，
当你没有想好函数的内容是可以用 pass 填充，使程序可以正常运行。

Python 解释器有两种运行模式
一个叫脚本 解释器腚眼一看 你这 .py 没有输入输出 直接全部执行就行 这个就叫脚本

另一个解释器定眼一看 你这有输出(光有输出不算) 解释器就会连接到系统知道的控制台进行交互
这个就叫做交互式运行环境

A. 交互式模式 (python)
  - 启动 PyRun_InteractiveLoop
  - 每次执行后检查返回值
  - 如果返回值不是 None，自动调用 sys.displayhook
  - displayhook 负责打印到 stdout

B. 脚本模式 (python script.py)
  - 启动 PyRun_SimpleFile
  - 执行整个文件
  - 忽略所有表达式的返回值
  - 除非显式调用 print()

  <子命令>
    rating          随机一题：只按 difficulty(rating) 区间筛选

    user            查询指定 handle 的 rating / max / rank 等一行摘要
    contests        列出进行中场次 + 即将开始的若干场（北京时间、距离开赛说明）

  user           & ".\.venv-msys\bin\python.exe" cf_service.py user tourist
