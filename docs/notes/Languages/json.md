json 就是一大堆键值对, 
然后他就是作为一种数据格式, 人看着方便, 计算机也是
```json 
{
  "variable_name": value(in int, string...),
  "array_name": [value_1, value_2],
  "dictionary_name": {
    "name": "shuaibi",
    "age": "666"
  }
}
```

json 经常在 python 和 js 中被读取并解析(但肯定不限于此)
